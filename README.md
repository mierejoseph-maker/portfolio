# Portfolio personnel — Joseph Miere

Portfolio web personnel de **Joseph Miere**, développeur web junior basé à Brazzaville. Cette page présente mon profil, ses compétences, son parcours, et mes moyens de contact dans une interface sobre, accessible et responsive.

## Aperçu

mon portfolio est une page unique comprenant :

- une présentation et des appels à l’action ;
- une section « À propos » avec quelques repères ;
- les compétences techniques et bureautiques ;
- une frise chronologique du parcours ;
- des notes de blog ;
- un formulaire de contact et des liens vers les réseaux sociaux ;
- un mode sombre activable sans JavaScript ;
- une navigation adaptée aux écrans mobiles.

## Technologies utilisées

- **HTML5** pour la structure et le contenu ;
- **CSS3** pour la mise en page, les animations, le thème sombre et le responsive design ;
- **Google Fonts** : Fraunces, Inter et JetBrains Mono.

## Structure du projet

```text
projet2/
├── portfolio.html  # Page principale du portfolio
├── portfolio.css   # Styles, responsive design et mode sombre
├── joseph.jpg      # Photo de profil
└── README.md       # Documentation du projet
```

## Lancer le projet

Aucune installation ni dépendance n’est nécessaire.

1. Téléchargez ou clonez le projet.
2. Ouvrez `portfolio.html` dans votre navigateur web.

## Personnalisation

Les contenus se modifient directement dans [portfolio.html](portfolio.html) :

- nom, titre et texte de présentation ;
- compétences et niveaux affichés ;
- expériences et articles ;
- adresse e-mail et liens sociaux ;
- photo de profil (`joseph.jpg`).

Pour modifier l’apparence, les couleurs ou les points de rupture mobile, éditez [portfolio.css](portfolio.css).

## Accessibilité et responsive design

Le site inclut un lien d’accès rapide au contenu principal, des libellés de formulaire, des textes alternatifs pour les images et une navigation pensée pour les petits écrans.
